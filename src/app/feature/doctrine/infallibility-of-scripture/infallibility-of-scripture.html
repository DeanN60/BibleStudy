The Old Testament was originally written in Hebrew. About 300 years before Christ, the Old Testament was translated to common Greek. And several centuries after Christ, the Old Testament was re-written in a more modern Hebrew called the Masoretic.

Most Bibles today use the Masoretic text for the Old Testament.

We have the Dead Sea Scrolls which are fragments of the original text found in the caves of Qumran next to the Dead Sea. We have the complete scroll of Isaiah from these scrolls which matches our current copies exactly.

The Hebrew alphabet has a numeric value for each letter. As the scribes copied the scrolls, they would sum up the value of the letters on a page of the copied text and check it against the sum of the value of the letters on the original page. If the new page did not match the original, they would use a scribes knife to cut out the new page.

The New Testament was originally written in common Greek and a few passages in Aramaic. This was done because most Jews at the time of Christ no longer spoke Hebrew.
